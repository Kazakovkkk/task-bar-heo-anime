Этап 1. Фундамент и Базовый движок (Текущий этап)
Цель: Запустить сердце игры в браузере, чтобы цифры начали правильно двигаться.

[] Создать структуру файлов (HTML, CSS, JS).

[] Написать простейший бесконечный цикл игры (нанесение урона монстру раз в секунду).

[ ] Реализовать ручной клик по монстру (чтобы игрок мог помогать персонажу).

[ ] Добавить базовое переключение между пассивным фармом (AFK) и активным боем.

Этап 2. Система персонажей (Аниме, Комиксы, Фильмы)
Цель: Оживить игру любимыми героями и настроить их особенности.

[ ] Спроектировать структуру данных для персонажей (JSON-база).

[ ] Добавить визуальное отображение (спрайты, GIF-анимации персонажей вместо смайликов).

[ ] Реализовать систему "Активного героя" (выбор, кто сейчас сражается).

[ ] Прописать уникальные пассивные способности (например, регенерация, критический удар, комбо).

Этап 3. Экономика, Магазин и Прогрессия
Цель: Сделать так, чтобы в игру было интересно возвращаться.

[ ] Создать магазин улучшений (покупка нового оружия/экипировки за золото).

[ ] Добавить систему уровней для самого персонажа (опыт за убийство монстров).

[ ] Сделать смену локаций/зон (каждые 10 монстров — босс, после которого открывается новая зона с более крутым лутом).

Этап 4. Переезд на Десктоп (Превращение в Task Bar приложение)
Цель: Вытащить игру из вкладки браузера и заставить её жить на панели задач Windows.

[ ] Настроить Tauri (или Electron) для сборки нашего кода в .exe файл.

[ ] Сделать окно игры прозрачным и убрать стандартные рамки Windows (закрыть/свернуть).

[ ] Написать скрипт автоматического позиционирования окна ровно над панелью задач (Taskbar).

[ ] Добавить иконку в системный трей (возле часов) с возможностью закрыть или свернуть игру.

Этап 5. Сохранения, Сок и Полировка (Juice)
Цель: Довести проект до ума, чтобы им было приятно пользоваться каждый день.

[ ] Реализовать систему сохранений (LocalStorage), чтобы прогресс не пропадал при закрытии программы.

[ ] Добавить "сочности" визуалу: всплывающие цифры урона (Floating Combat Text), кастомные шрифты.

[ ] Настроить оффлайн-прогресс (игра считает, сколько золота принес герой, пока компьютер был выключен или программа была закрыта).
