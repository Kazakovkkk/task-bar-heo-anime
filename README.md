# Task Bar Hero Anime

Idle/clicker-игра в духе Task Bar Hero, но с anime-inspired персонажами. Сначала делаем браузерный TypeScript-прототип, затем переносим его в desktop overlay над панелью задач Windows.

## Core Loop

Герои автоматически идут справа налево -> монстры идут слева направо -> отряды встречаются на арене -> начинается авто-бой -> игрок помогает кликами по монстрам -> монстр умирает -> игрок получает золото и опыт -> покупает улучшения -> герои становятся сильнее -> появляются более сложные волны.

## Visual Direction

Цель по визуалу: широкая side-view пиксельная сцена, похожая на референс с дорожкой, HP-барами над юнитами и несколькими существами на поле.

- Герои появляются справа и сами двигаются влево.
- Монстры появляются слева и двигаются вправо.
- Когда юниты сближаются, они останавливаются и атакуют.
- Игрок кликает по монстрам, чтобы наносить дополнительный урон.
- Текущие спрайты CSS-placeholder; позже заменим их на настоящие pixel-art sprites/GIF.

## How to Run

```bash

```


## Initial Balance

Стартовые формулы можно менять после теста, но для первого TypeScript-прототипа используем простые правила:

- HP монстра: `12 * 1.18 ^ (monsterLevel - 1)`.
- Награда золотом: `5 * 1.12 ^ (monsterLevel - 1)`.
- Урон клика: `1 + heroLevel * 0.25 + clickUpgradeLevel`.
- DPS героя: `2 + heroLevel * 0.45 + swordLevel * 0.85`.
- Стоимость улучшения меча: `10 * 1.25 ^ swordLevel`.
- Стоимость улучшения клика: `8 * 1.25 ^ clickUpgradeLevel`.
- Каждый 10-й монстр становится боссом с увеличенным HP, уроном и наградой.
